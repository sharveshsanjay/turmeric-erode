# ERODE | The Turmeric Capital of India

An interactive documentary experience exploring the world-famous turmeric of Erode, Tamil Nadu. Follow the journey from fertile soil to millions of homes around the globe.

**Live Demo:** [turmeric-erode.vercel.app](https://turmeric-erode.vercel.app/)

---

## 📖 About

ERODE is an immersive digital documentary that celebrates the rich heritage and agricultural significance of Erode's turmeric industry. Through interactive storytelling and visual narratives, this project brings to life the stories of farmers, the cultivation process, and the global impact of one of India's most precious commodities.

---

## ✨ Features

- 🌾 **Interactive Documentary** - Engage with the turmeric cultivation story through an immersive web experience
- 🌍 **Global Impact Visualization** - Discover how Erode's turmeric reaches millions of homes worldwide
- 📱 **Responsive Design** - Optimized for desktop, tablet, and mobile devices
- 🎨 **Rich Visual Content** - Beautiful imagery and animations that bring the story to life
- 🔍 **Educational** - Learn about traditional farming methods and modern agricultural practices
- ⚡ **Fast Performance** - Deployed on Vercel for optimal loading speeds

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/sharveshsanjay/turmeric-erode.git
cd turmeric-erode
```

### 2. Install Dependencies

```bash
npm install
# or
yarn install
```

### 3. Run the Development Server

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

### 4. Build for Production

```bash
npm run build
npm start
# or
yarn build
yarn start
```
